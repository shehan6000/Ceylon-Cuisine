# Ceylon-Cuisine
This project is a website dedicated to showcasing a variety of Sri Lankan recipes. Explore traditional and modern dishes, beverages, desserts, and more.

## Project Structure

- **assets/**: Contains images and other assets.
- **css/**: Contains CSS files for styling the website.
- **js/**: Contains JavaScript files for interactive functionality.
- **recipes/**: Directory for storing individual recipe files.
- **404.html**: Custom 404 error page.
- **about.html**: Information about the website.
- **beverages.html**: Recipes for Sri Lankan beverages.
- **contact.html**: Contact information page.
- **desserts.html**: Recipes for Sri Lankan desserts.
- **dishes.html**: Recipes for various Sri Lankan dishes.
- **index.html**: Homepage of the website.
- **sweets.html**: Recipes for Sri Lankan sweets.

## How to Navigate

- Visit the [Homepage](index.html) to start exploring.
- Check out [About Us](about.html) to learn more about the website.
- Browse recipes by category:
  - [Beverages](beverages.html)
  - [Desserts](desserts.html)
  - [Dishes](dishes.html)
  - [Sweets](sweets.html)
- Use the [Contact Us](contact.html) page to get in touch.
